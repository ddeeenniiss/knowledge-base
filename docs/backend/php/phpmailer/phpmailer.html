PHP Mailer

Grundlegendes bei Redaxo:

Das AddON bei Redaxo wird in 3 Teile aufgeteilt: Konfifuration, Email-Log und Archiv

Konfiguration

Hier werden generelle Einstellungen getätigt. Ansich trivial. Mailertyp wird dann unterschieden in mail, smtp, sendmail

smtp benötigt Host, Port (Standard SMTP 25), Verschlüsselung, Verschlüsselungstyp, Authentifizierung, Benutzername und Passwort.
