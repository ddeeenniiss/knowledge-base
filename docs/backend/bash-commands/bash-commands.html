Bash-Kommandos

Grundlegende Befehle

ls - Listet Dateien und Verzeichnisse im aktuellen Verzeichnis auf. (list)

cd - Wechselt das aktuelle Verzeichnis. (change directory)

pwd - Zeigt den aktuellen Verzeichnispfad an. (print working directory)

mkdir - Erstellt ein neues Verzeichnis. (make directory)

rm - Löscht Dateien oder Verzeichnisse. (remove)

cp - Kopiert Dateien oder Verzeichnisse. (copy)

mv - Verschiebt oder benennt Dateien oder Verzeichnisse um. (move)

Datei- und Verzeichnisoperationen

touch - Erstellt eine neue leere Datei oder aktualisiert den Zeitstempel einer Datei.

cat - Zeigt den Inhalt einer Datei an. (concatenate)

nano - Ein einfacher Texteditor im Terminal.

vim - Ein leistungsfähiger Texteditor im Terminal.

head - Zeigt die ersten Zeilen einer Datei an.

tail - Zeigt die letzten Zeilen einer Datei an.

Pipelines

grep - Sucht nach Textmustern in Dateien. (global regular expression print)

sort - Sortiert die Zeilen einer Datei.

uniq - Entfernt doppelte aufeinanderfolgende Zeilen aus einer Datei.

wc - Zählt die Anzahl der Zeilen, Wörter und Zeichen in einer Datei. (word count)
