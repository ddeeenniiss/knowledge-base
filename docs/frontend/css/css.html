property: zB transform, all
duration: wie lange dauert die Transition, zB 2s
timing-function: linear konstante Geschwindigkeit cubic-bezier(0, 0, 1, 1) ease
(default) langsam → schnell → langsam cubic-bezier(0.25, 0.1, 0.25, 1) ease-in startet langsam ease-in =

cubic-bezier: (x1, y1, x2, y2) 
Eine Timing-Function beschreibt nicht die Bewegung im Raum, sondern: Wie schnell der Fortschritt

(0,0) Endpunkt ist immer (1,1) Beispiel: cubic-bezier(0.25, 0.1, 0.25, 1) x1 = 0.25 bedeutet sehr früher Einfluss der

Pseudo-Elemente


::before und ::after


::before und ::after ermöglichen das Einfügen von Inhalten vor oder nach dem Inhalt eines Elements.
Sie werden oft für dekorative Zwecke oder zusätzliche Informationen verwendet, ohne das HTML zu verändern.


.element::before {
  content: "Vor dem Inhalt";
  color: blue;
}
.element::after {
  content: "Nach dem Inhalt";
  color: red;
}



::backdrop


::backdrop ist ein Pseudo-Element, das auf das Hintergrund-Element eines Modals oder Popups angewendet wird. zB Hintergrund verdunkeln hinter Popup.


.modal::backdrop {
  background-color: rgba(0, 0, 0, 0.5);
}

Pseudo-Klassen


:hover


:hover ist ein Pseudo-Klassen-Selektor, der angewendet wird, wenn der Benutzer den Mauszeiger über ein Element bewegt.


.element:hover {
  background-color: yellow;
}



:focus


:focus ist ein Pseudo-Klassen-Selektor, der angewendet wird, wenn ein Element den Fokus erhält, z.B. durch Klicken oder Tabben.


.element:focus {
  outline: 2px solid blue;
}



:focus-visible


:focus-visible ist ein Pseudo-Klassen-Selektor, der angewendet wird, wenn ein Element den Fokus erhält und der Fokus sichtbar ist (z.B. durch Tastatur-Navigation).


.element:focus-visible {
  outline: 2px solid blue;
}



:active


:active ist ein Pseudo-Klassen-Selektor, der angewendet wird, wenn ein Element aktiv ist, z.B. beim gedrückten Klick, .


.element:active {
  background-color: green;
}



:checked


:checked ist ein Pseudo-Klassen-Selektor, der angewendet wird, wenn ein Element (z.B. ein Checkbox oder Radio Button) aktiviert ist.


.checkbox:checked + .label {
  font-weight: bold;
}
