Basics

CSS Grid ist ein zweidimensionales Layout-System für das Web.

Typische Einsatzfälle sind: Seitenlayouts (Header / Sidebar / Content / Footer), Dashboards, Galerien & Karten-Layouts, Komplexe responsive Layouts ohne Hacks

Grundprinzip:

Der Grid Container ist das Elternelement und die Grid Items sind die direkten Kindselemente. Nur direkte Kindselemente werden auch zu Grid Items.

display: grid;

Spalten und Zeilen definieren:

Das wird festgelegt mit 

grid-template-columns: 1fr 1fr 200px;
und 
grid-template-rows: 1fr 1fr 200px;
. Die Einheiten sind dabei px für genaue Werte, % für Anteile und fr für verfügbaren space. Bei auto bestimmt der Inhalt jeweils die Größe.

Abstand zwischen Items

Das wird gesetzt entweder mit 

row-gap: 20px;
für Abstand zwischen Reihen. Andernfalls mit 
column-gap: 20px;
für Abstand zwischen Spalten. Beides kombiniert kann auch mit 
gap: 20px;
angegeben werden für Reihen und Spalten.

Grid Items Platzierung

Feste Platzierung eines Grid Items kann man setzen mit 

grid-column: column-start / column-end;
auf dem Kindselement. Das bedeutet das Kindselement geht von Spalte column-start bis Spalte column-end. Mit 
grid-row: row-start / row-end;
kann der Reihenstart row-start bis Reihenende row-end festgelegt werden. In kombinierter Schreibweise lässt sich auch 
grid-area: row-start / col-start / row-end / col-end;
festlegen.

repeat() Abkürzungsfunktion

Mit der 

grid-template-columns: repeat(anzahl, größe);
Funktion lassen sich mehrere gleichartige Zeilen oder Spalten definieren, ohne diese einzeln aufzuschreiben.

repeat() mit minmax() Funktion

Die 

minmax(minValue, maxValue)
Funktion lässt ein Kindselement minimal minValue und maximal maxValue groß werden. Dazwischen reagiert es flexibel je nach Container Größe.

repeat() mit autofit

grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
wird verwendet, damit die Spalten sich automatisch an den verfügbaren Platz anpassen. Bsp: Erzeuge so viele Spalten wie möglich und lass sie den restlichen Platz ausfüllen.

Grid Areas - Layouts lesbar machen

Mit 

grid-template-areas: "header header"
                                    "sidebar main"
                                    "footer footer";
lassen sich Grids definieren. Diese müssen dann noch per 
.header  { grid-area: header; }
            .sidebar{ grid-area: sidebar; }
            .main    { grid-area: main; }
            .footer  { grid-area: footer; }
zugewiesen werden.

Grid Subgrid

Mit 

grid-template-columns: subgrid;
lassen sich dieselben Zeilen und Spalten verwenden wie auch beim parent Element. Die Eigenschaft wird dem child jeweils zugewiesen und wirkt nur auf direkte Kindselemente.
