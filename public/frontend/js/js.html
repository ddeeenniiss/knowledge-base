IntersectionObserver

Der IntersectionObserver ermöglicht es, das Sichtbarkeitsverhalten von Elementen im Viewport zu überwachen. Dies ist besonders nützlich für Lazy Loading von Bildern, Infinite Scrolling oder das Auslösen von Animationen, wenn ein Element in den sichtbaren Bereich des Bildschirms gelangt.

Beispiel: Module bei straehle, die in den Bildschirm einfaden

    document.addEventListener('DOMContentLoaded', () => {
  const elements = document.querySelectorAll('.animate-on-scroll');

  const observer = new IntersectionObserver(
    (entries, obs) => {
      entries.forEach(entry => {
        if (entry.isIntersecting) {
          entry.target.classList.add('is-visible');
          obs.unobserve(entry.target); // nur einmal animieren
        }
      });
    },
    {
      threshold: 0.1 // 10% sichtbar = Animation starten
    }
  );

  elements.forEach(el => observer.observe(el));
});

DOMContentLoaded

Das DOMContentLoaded-Ereignis tritt auf, sobald der HTML-Dokument-Inhalt vollständig geladen und geparst wurde, ohne auf externe Ressourcen wie Bilder oder Skripte zu warten. Es ist nützlich, um sicherzustellen, dass das DOM (zB. Buttons, Listen, Nav Elemente) bereit ist, bevor JavaScript-Operationen ausgeführt werden.

document.addEventListener('DOMContentLoaded', () => {});

DOM (Document Object Model)

ist eine Schnittstelle, um den Inhalt, die Struktur und das Styling eines HTML-Dokuments dynamisch zu verändern. Es repräsentiert das Dokument als Baumstruktur, wobei jedes Element im Baum ein Objekt ist, das über JavaScript manipuliert werden kann. Der Einstiegspunkt dafür ist .document

Event Listener

Ein Event Listener ist eine Funktion, die auftritt, wenn ein bestimmtes Ereignis (z.B. Klick, Hover) auf einem Element passiert. Sie ermöglicht es, auf Benutzerinteraktionen zu reagieren und dynamische Aktionen auszuführen.

element.addEventListener('click', function() {
  // Aktion bei Klick
});

Beispiel: Button Klick Event Listener

document.addEventListener('DOMContentLoaded', () => {
  const button = document.getElementById('myButton');
  button.addEventListener('click', () => {
    alert('Button wurde geklickt!');
  });
});

getElementById() ist eine Methode des DOM, um ein Element anhand seiner ID zu finden.

getElementByClassName() ist eine Methode des DOM, um Elemente anhand ihres Klassennamens zu finden.

querySelector() ist eine Methode des DOM, um das erste Element zu finden, das einem CSS-Selektor entspricht.

querySelectorAll() ist eine Methode des DOM, um alle Elemente zu finden, die einem CSS-Selektor entsprechen.

getElementByTagName() ist eine Methode des DOM, um alle Elemente zu finden, die einem HTML-Tagnamen entsprechen.

Schlüsselwort return

return verlässt die Funktion sofort.

zB if (!button) return; // Funktion verlassen, wenn Button nicht gefunden wurde (Guard Clause). Dann wird restlicher Code garnicht erst ausgeführt. Dadurch werden Fehler vermieden, wenn das Element nicht da ist.
