PHP Basics

PHP verwendet <?php ... ?> Tags, um PHP-Code in HTML-Dateien einzubetten.

Idealerweise lässt sich <?= nutzen, der eine Kurzform für <?php echo darstellt.

echo 'Hallo Welt'; dient zum Ausgeben von Text ('Text') oder Variablen ($variable) auf der Webseite. Ein Semikolon ist hierbei zu verwenden.

Kommentare in PHP werden mit // für einzeilige Kommentare oder /* ... */ für mehrzeilige Kommentare erstellt.

Variablen

Variablen in PHP beginnen mit einem $-Zeichen, gefolgt vom Variablennamen. 

Beispiel: $name = "John";

Die Typen sind dabei dynamisch, daher kein integer, string, float etc. notwendig.

Variablen verketten:

Um Variablen in PHP zu verketten, verwendet man den Punkt (.) als Verknüpfungsoperator. 

Beispiel: echo $name . " " . $lastname;

Arrays

Arrays in PHP werden mit dem array()-Konstruktur oder der Kurzsyntax [] erstellt.

Beispiel: $fruits = array("Apfel", "Banane", "Orange");

oder

$fruits = ["Apfel", "Banane", "Orange"];

Der Zugriff auf Array-Elemente erfolgt über den Index

z.B. $fruits[0] für "Apfel".

Ebenso gibt es assoziative Arrays, bei denen Schlüssel benannt werden

z.B. $person = ["name" => "John", "age" => 30];

Der Zugriff hier erfolgt über den Schlüssel, z.B. $person["name"] für "John".

Bedingungen

Bedingungen in PHP werden mit if, elseif und else abgefragt. Beispiel: if ($age >= 18) { echo "Du bist volljährig"; } else { echo "Du bist minderjährig"; }

Vergleichsoperatoren sind z.B. ==, !=, <, >, <=, >=. ===, wenn exakt gleicher Typ und Wert verglichen werden sollen.

Schleifen

Schleifen in PHP können mit for, while und foreach erstellt werden. Beispiel für eine for-Schleife: for ($i = 0; $i < 10; $i++) { echo $i; }

Beispiel für eine foreach-Schleife: foreach ($fruits as $fruit) { echo $fruit; }

Funktionen

Funktionen in PHP werden mit dem function-Schlüsselwort definiert. Beispiel: function greet($name) { return "Hallo " . $name; }

Funktionen können Parameter entgegennehmen und Werte zurückgeben.

Includes / Requires:

include und require werden verwendet, um externe PHP-Dateien einzubinden. Beispiel: include 'header.php'; oder require 'config.php';

Der Unterschied besteht darin, dass require einen fatalen Fehler auslöst, wenn die Datei nicht gefunden wird, während include nur eine Warnung ausgibt.

Superglobale Variablen

PHP bietet superglobale Variablen wie $_GET, $_POST, $_SESSION, $_SERVER, $_FILES und $_COOKIE, um auf Daten aus Formularen, Sitzungen und Cookies zuzugreifen.

Beispiel: $name = $_POST['name']; um auf einen Wert aus einem POST-Formular zuzugreifen.

Beispiel: $name = $_GET['name']; um auf einen Wert aus einem GET-Formular zuzugreifen.

Beispiel: $_SESSION['user'] = "John"; um eine Sitzung zu starten oder auf eine Sitzung zuzugreifen.

Beispiel: $_SERVER['HTTP_USER_AGENT']; um Informationen über den Browser des Benutzers zu erhalten.

Beispiel: $_FILES['file']['name']; um auf den Namen einer hochgeladenen Datei zuzugreifen.

Beispiel: $_COOKIE['user']; um auf einen Wert aus einem Cookie zuzugreifen.
